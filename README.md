# Project 1: SPA Landing Page

A simple, elegant single-page application landing page with a modern gradient design.

## Tech Stack

- HTML5
- CSS3 (embedded styles)
- Pure vanilla JavaScript (SPA structure)

## Features

- Clean, minimalist design
- Gradient background (purple/blue theme)
- Responsive layout
- Hover animations on buttons and links
- Contact CTA button
- Social media links (GitHub, LinkedIn, Twitter)

## Setup

No build process required. Simply open the HTML file in a browser:

```bash
open index.html
# or
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Customization

Edit `index.html` to customize:
- Name and tagline (lines 83-84)
- Email address (line 85)
- Social media links (lines 87-89)
- Colors in the `<style>` section (line 20 for gradient)
